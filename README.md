# Prosthetic Access Atlas

An open-access resource mapping prosthetic/orthotic care access gaps and clinical trial landscapes to improve prosthetic care equity in underserved regions of the United States.

## Overview

This project combines two data streams: Clinical Trial Landscape analysis and Coverage Gap Analysis for three underserved US regions.

## Key Findings

- **Rural West Virginia (Charleston)**: 0 prosthetic/orthotic providers within 100km
- **Eastern Kentucky (Jackson)**: 0 prosthetic/orthotic providers within 100km
- **Mississippi Delta (Leland)**: 0 prosthetic/orthotic providers within 100km

## Structure

- `clinical-trials.md` — Clinical trial trends and analysis
- `gap-analysis.md` — Regional coverage gap maps and provider inventories
- `data/sources.json` — Raw data sources and region coordinates
